# Projeto – Entrega I (HTML5)

Projeto desenvolvido para a entrega da Unidade, contendo:

- Estrutura HTML5 semântica
- 3 páginas obrigatórias: index, projetos e cadastro
- Formulário com validação nativa
- Máscaras de CPF, telefone e CEP
- Pastas organizadas (css, js, images)

## Como visualizar

Basta abrir o arquivo `index.html` no navegador.
